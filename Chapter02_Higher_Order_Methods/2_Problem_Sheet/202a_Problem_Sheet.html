Problem Sheet 2 Question 2a - 3rd Order Taylor¶

The general form of the population growth differential equation

(199)¶\[\begin{equation}y^{'}=t-y, \ \ (0 \leq t \leq 4),\end{equation}\]

with the initial condition

(200)¶\[\begin{equation}y(0)=1.\end{equation}\]

For N=4 with the analytic (exact) solution

(201)¶\[\begin{equation} y= 2e^{-t}+t+1.\end{equation}\]

Apply 3rd Order Taylor Method to approximate the solution of the given initial value problems using the indicated number of time steps. Compare the approximate solution with the given exact solution, and compare the actual error with the theoretical error.

General Discrete Interval¶

The continuous time \(a\leq t \leq b \) is discretised into \(N\) points seperated by a constant stepsize

(202)¶\[\begin{equation} h=\frac{b-a}{N}.\end{equation}\]

Specific Discrete Interval¶

Here the interval is \(0\leq t \leq 4\) with \(N=4\)

(203)¶\[\begin{equation} h=\frac{4-0}{4}=1.\end{equation}\]

This gives the 5 discrete points with stepsize h=1:

(204)¶\[\begin{equation} t_0=0, \ t_1=1, \ ... t_{4}=4. \end{equation}\]

This is generalised to

(205)¶\[\begin{equation} t_i=0+i1, \ \ \ i=0,1,...,4.\end{equation}\]

The plot below illustrates the discrete time steps from 0 to 4.

### Setting up time

3rd Order Taylor Solution¶

The 3rd Order Taylor difference equation is given by

(206)¶\[\begin{equation}w_{i+1}=w_i+h\left[f(t_i,w_i)+\frac{h}{2}f'(t_i,w_i)+\frac{h^2}{6}f''(t_i,w_i)\right],\end{equation}\]

where

(207)¶\[\begin{equation} f(t_i,w_i)=t_i-w_i,\end{equation}\]
(208)¶\[\begin{equation}f'(t_i,w_i)=1-t_i+w_i,\end{equation}\]

and

(209)¶\[\begin{equation} f''(t_i,w_i)=-1+t_i-w_i,\end{equation}\]

which gives

(210)¶\[\begin{equation} w_{i+1}=w_i+h(t_i-w_i)+\frac{h^2}{2}(1-t_i+w_i)+\frac{h^3}{6}(-1+t_i-w_i), \end{equation}\]

for \(i=0,1,2,3\) with the initial condition,

(211)¶\[\begin{equation}w_0=1.\end{equation}\]
IC=1

INTITIAL_CONDITION=IC
w=np.zeros(N+1)
w[0]=INTITIAL_CONDITION
for i in range (0,N):
    w[i+1]=w[i]+h*(t[i]-w[i])+h*h/2*(1-t[i]+w[i])+h*h*h/6*(-1+t[i]-w[i])

y=2*np.exp(-t)+t-1 # Exact Solution

Global Error¶

The upper bound of the global error is:

(212)¶\[\begin{equation} |y(t_i)-w_i|\leq\frac{Mh^3}{24}\frac{1}{L} |e^{Lt_i}-1| \end{equation}\]

where \(|y''''(t)|\leq M \) for \(t \in (0,4)\) and \(L\) is the Lipschitz constant. Generally we do not have access to \(M\) or \(L\), but as we have the exact solution we can find \(M\) and \(L\).

Local Error¶

The global error consistent of an exponential term and the local error. The local error for the 3rd order Taylor method is

(213)¶\[\begin{equation} |y(t_1)-w_1|<\tau=\frac{Mh^3}{24} \end{equation}\]

by differentiation the exact solution

(214)¶\[\begin{equation} y= 2e^{-t}+t+1\end{equation}\]

four times

(215)¶\[\begin{equation}y'=-2e^{-t}+1,\end{equation}\]
(216)¶\[\begin{equation}y''=2e^{-t},\end{equation}\]
(217)¶\[\begin{equation}y'''=-2e^{-t},\end{equation}\]

and

(218)¶\[\begin{equation}y''''=2e^{-t}.\end{equation}\]

Then we choose the largest possible value of \(y'''\) on the interval of interest \((0,4)\), which gives,

(219)¶\[\begin{equation}|y''(t)|\leq |2e^{0}|=2, \end{equation}\]

therefore

(220)¶\[\begin{equation}M=2.\end{equation}\]

This gives the local error

(221)¶\[\begin{equation} |y(t_1)-w_1|<\tau=\frac{Mh^3}{24}=\frac{2h^3}{24}=\frac{h^3}{12}.\end{equation}\]

Lipschitz constant¶

The Lipschitz constant \(L\) is from the Lipschitz condition,

(222)¶\[\begin{equation}\left| \frac{\partial f(t,y)}{\partial t}\right|\leq L. \end{equation}\]

The constant can be found by taking partical derivative of \(f(t,y)=t-y\) with respect to \(y\)

(223)¶\[\begin{equation}\frac{\partial f(t,y)}{\partial y}=-1\end{equation}\]
(224)¶\[\begin{equation}L=|-1|=1.$$ ### Specific Global Upper Bound Formula Puting these values into the upper bound gives: \begin{equation} |y(t_i)-w_i|\leq\frac{h^3}{12} |e^{t_i}-1|. \end{equation}\]
Upper_bound=h*h*h/(12)*(np.exp(t)-1) # Upper Bound
