Problem Sheet 2 Question 1b - 2nd Order Taylor¶

The general form of the population growth differential equation

(120)¶\[\begin{equation} y^{'}=y-t, \ \ (0 \leq t \leq 2) \end{equation}\]

with the initial condition

(121)¶\[\begin{equation}y(0)=2.\end{equation}\]

For N=4 with the analytic (exact) solution

(122)¶\[\begin{equation} y= e^{t}+t+1.\end{equation}\]

Apply _2nd Order Taylor to approximate the solution of the given initial value problems using the indicated number of time steps. Compare the approximate solution with the given exact solution, and compare the actual error with the theoretical error.

General Discrete Interval¶

The continuous time \(a\leq t \leq b \) is discretised into \(N\) points seperated by a constant stepsize

(123)¶\[\begin{equation} h=\frac{b-a}{N}.\end{equation}\]

Specific Discrete Interval¶

Here the interval is \(0\leq t \leq 2\) with \(N=4\)

(124)¶\[\begin{equation} h=\frac{2-0}{4}=0.5.\end{equation}\]

This gives the 5 discrete points with stepsize h=0.5:

(125)¶\[\begin{equation} t_0=0, \ t_1=0.5, \ ... t_{4}=2. \end{equation}\]

This is generalised to

(126)¶\[\begin{equation} t_i=0+i0.5, \ \ \ i=0,1,...,4.\end{equation}\]

The plot below illustrates the discrete time steps from 0 to 2.

### Setting up time
t_end=2

N=4

2nd Order Taylor Solution¶

The 2nd Order Taylor difference equation is given by

(127)¶\[\begin{equation} w_{i+1}=w_i+h\left[f(t_i,w_i)+\frac{h}{2}f'(t_i,w_i)\right],\end{equation}\]

where

(128)¶\[\begin{equation} f(t_i,w_i)=w_i-t_i,\end{equation}\]

and

(129)¶\[\begin{equation} f'(t_i,w_i)=w_i-t_i-1,\end{equation}\]

which gives

(130)¶\[\begin{equation} w_{i+1}=w_i+h(w_i-t_i)+\frac{h^2}{2}(w_i-t_i-1), \end{equation}\]

for \(i=0,1,2,3\) with \(w_0=2.\)

IC=2

INTITIAL_CONDITION=IC
w=np.zeros(N+1)
w[0]=INTITIAL_CONDITION
for i in range (0,N):
    w[i+1]=w[i]+h*(w[i]-t[i])+h*h/2*(w[i]-t[i]-1)

y=np.exp(t)+t+1 # Exact Solution

Global Error¶

The upper bound of the global error is:

(131)¶\[\begin{equation} |y(t_i)-w_i|\leq\frac{Mh^2}{6}\frac{1}{L} |e^{Lt_i}-1| \end{equation}\]

where \(|y'''(t)|\leq M \) for \(t \in (0,2)\) and \(L\) is the Lipschitz constant. Generally we do not have access to \(M\) or \(L\), but as we have the exact solution we can find \(M\) and \(L\).

Local Error¶

The global error consistent of an exponential term and the local error. The local error for the 2nd order Taylor method is

(132)¶\[\begin{equation} |y(t_1)-w_1|<\tau=\frac{Mh^2}{6} $$ by differentiation the exact solution \begin{equation} y= e^{t}+t+1,\end{equation}\]

thrice

(133)¶\[\begin{equation}y'= e^{t}+1,\end{equation}\]
(134)¶\[\begin{equation}y''=e^{t},\end{equation}\]

and

(135)¶\[\begin{equation}y'''=e^{t}.\end{equation}\]

Then we choose the largest possible value of \(y'''\) on the interval of interest \((0,2)\), which gives,

(136)¶\[\begin{equation}|y''(t)|\leq e^{2}=7.38<8, \end{equation}\]

therefore

(137)¶\[\begin{equation}M=8.\end{equation}\]

This gives the local error

(138)¶\[\begin{equation} |y(t_1)-w_1|<\tau=\frac{Mh^2}{6}=\frac{8h^2}{6}.\end{equation}\]

Lipschitz constant¶

The Lipschitz constant \(L\) is from the Lipschitz condition,

(139)¶\[\begin{equation}\left| \frac{\partial f(t,y)}{\partial t}\right|\leq L. \end{equation}\]

The constant can be found by taking partical derivative of \(f(t,y)=y-t\) with respect to \(y\)

(140)¶\[\begin{equation}\frac{\partial f(t,y)}{\partial y}=1,\end{equation}\]
(141)¶\[\begin{equation}L=1.\end{equation}\]

Specific Global Upper Bound Formula¶

Puting these values into the upper bound gives:

(142)¶\[\begin{equation} |y(t_i)-w_i|\leq\frac{8h^2}{6} |e^{t_i}-1|. \end{equation}\]
Upper_bound=8*h*h/(6*1)*(np.exp(t)-1) # Upper Bound
