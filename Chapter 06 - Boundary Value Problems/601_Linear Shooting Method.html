Introduction¶

To numerically approximate the Boundary Value Problem

(551)¶\[\begin{equation} y^{''}=p(x)y^{'}+q(x)y+r(x) \ \ \ a < x < b \end{equation}\]

with the left and right boundary conditions:

(552)¶\[\begin{equation}y(a)=\alpha, \end{equation}\]
(553)¶\[\begin{equation}y(b) =\beta. \end{equation}\]

The Boundary Value Problem is divided into two Initial Value Problems:

  1. The first 2nd order Initial Value Problem is the same as the original Boundary Value Problem with an extra initial condtion \(y_1^{'}(a)=0\).

(554)¶\[\begin{equation} y^{''}_1=p(x)y^{'}_1+q(x)y_1+r(x), \ \ y_1(a)=\alpha, \ \ \color{green}{y^{'}_1(a)=0},\\ \end{equation}\]

  1. The second 2nd order Initial Value Problem is the homogenous form of the original Boundary Value Problem, by removing \(r(x)\), with the initial condtions \(y_2(a)=0\) and \(y_2^{'}(a)=1\).

(555)¶\[\begin{equation} y^{''}_2=p(x)y^{'}_2+q(x)y_2, \ \ \color{green}{y_2(a)=0, \ \ y^{'}_2(a)=1}. \end{equation}\]

combining these intial values problems together to get the unique solution

(556)¶\[\begin{equation} y(x)=y_1(x)+\frac{\beta-y_1(b)}{y_2(b)}y_2(x), \end{equation}\]

provided that \(y_2(b)\not=0\).

The truncation error for the shooting method is
$\( |y_i - y(x_i)| \leq K h^n\left|1+\frac{y_{2 i}}{y_{2 i}}\right| \)\( \)O(h^n)$ is the order of the numerical method used to approximate the solution of the Initial Value Problems.

Example Boundary Value Problem¶

To illustrate the shooting method we shall apply it to the Boundary Value Problem:

(557)¶\[\begin{equation}y^{''}=2y^{'}+3y-6, \end{equation}\]

with boundary conditions

(558)¶\[\begin{equation}y(0) = 3, \end{equation}\]
(559)¶\[\begin{equation}y(1) = e^3+2, \end{equation}\]

with the exact solution is

(560)¶\[\begin{equation}y=e^{3x}+2. \end{equation}\]

The boundary value problem is broken into two second order Initial Value Problems:

  1. The first 2nd order Intial Value Problem is the same as the original Boundary Value Problem with an extra initial condtion \(u^{'}(0)=0\).

(561)¶\[\begin{equation} u^{''} =2u'+3u-6, \ \ \ \ u(0)=3, \ \ \ \color{green}{u^{'}(0)=0} \end{equation}\]

  1. The second 2nd order Intial Value Problem is the homogenous form of the original Boundary Value Problem with the initial condtions \(w^{'}(0)=0\) and \(w^{'}(0)=1\).

(562)¶\[\begin{equation} w^{''} =2w^{'}+3w, \ \ \ \ \color{green}{w(0)=0}, \ \ \ \color{green}{w^{'}(0)=1} \end{equation}\]

combining these results of these two intial value problems as a linear sum

(563)¶\[\begin{equation} y(x)=u(x)+\frac{e^{3x}+2-u(1)}{w(1)}w(x) \end{equation}\]

gives the solution of the Boundary Value Problem.

Discrete Axis¶

The stepsize is defined as

(564)¶\[\begin{equation}h=\frac{b-a}{N}, \end{equation}\]

here it is

(565)¶\[\begin{equation}h=\frac{1-0}{10}, \end{equation}\]

giving

(566)¶\[\begin{equation}x_i=0+0.1 i, \end{equation}\]

for \(i=0,1,...10.\)

Numerical method¶

The Euler method is applied to numerically approximate the solution of the system of the two second order initial value problems they are converted in to two pairs of two first order initial value problems:

1. Inhomogenous Approximation¶

The plot below shows the numerical approximation for the two first order Intial Value Problems

(567)¶\[\begin{equation} u_1^{'} =u_2, \ \ \ \ u_1(0)=3, \end{equation}\]
(568)¶\[\begin{equation} u_2^{'} =2u_2+3u_1-6, \ \ \ \color{green}{u_2(0)=0}, \end{equation}\]

that Euler approximate of the inhomogeneous two Initial Value Problems is :

(569)¶\[\begin{equation}u_{1}[i+1]=u_{1}[i] + h u_{2}[i] \end{equation}\]
(570)¶\[\begin{equation}u_{2}[i+1]=u_{2}[i] + h (2u_{2}[i]+3u_{1}[i] -6) \end{equation}\]

with \(u_1[0]=3\) and \(\color{green}{u_2[0]=0}\).

for i in range (0,N):
    U1[i+1]=U1[i]+h*(U2[i])
    U2[i+1]=U2[i]+h*(2*U2[i]+3*U1[i]-6)

2. Homogenous Approximation¶

The homogeneous Bounday Value Problem is divided into two first order Intial Value Problems

(571)¶\[\begin{equation} w_1^{'} =w_2, \ \ \ \ \color{green}{w_1(1)=0} \end{equation}\]
(572)¶\[\begin{equation} w_2^{'} =2w_2+3w_1, \ \ \ \color{green}{w_2(1)=1} \end{equation}\]

The Euler approximation of the homogeneous of the two Initial Value Problem is

(573)¶\[\begin{equation}w_{1}[i+1]=w_{1}[i] + h w_{2}[i] \end{equation}\]
(574)¶\[\begin{equation}w_{2}[i+1]=w_{2}[i] + h (2w_{2}[i]+3w_{1}[i]) \end{equation}\]

with \(\color{green}{w_1[0]=0}\) and \(\color{green}{w_2[1]=1}\).

for i in range (0,N):
    W1[i+1]=W1[i]+h*(W2[i])
    W2[i+1]=W2[i]+h*(2*W2[i]+3*W1[i])

Approximate Solution¶

Combining together the numerical approximation of \(u_1\) and \(w_1\) as a weighted sum

(575)¶\[\begin{equation}y(x[i])\approx u_{1}[i] + \frac{e^3+2-u_{1}[N]}{w_1[N]}w_{1}[i] \end{equation}\]

gives the approximate solution of the Boundary Value Problem.

The truncation error for the shooting method using the Euler method is

(576)¶\[\begin{equation} |y_i - y(x[i])| \leq K h\left|1+\frac{w_{1}[i]}{u_{1}[i]}\right| \end{equation}\]

\(O(h)\) is the order of the method.
