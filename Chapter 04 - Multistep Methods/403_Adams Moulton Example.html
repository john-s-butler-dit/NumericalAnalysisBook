Adams Moulton¶

The Adams Moulton method is an implicit multistep method. This notebook illustrates the 2 step Adams Moulton method for a linear initial value problem of the form

(375)¶\[\begin{equation} y^{'}=t-y, \ \ (0 \leq t \leq 2)\end{equation}\]

with the initial condition

(376)¶\[\begin{equation}y(0)=1.\end{equation}\]

def myfun_ty(t,y):

Discrete Interval¶

Defining the step size \(h\) from the interval range \(a\leq t \leq b\) and number of steps \(N\)

(378)¶\[\begin{equation}h=\frac{b−a}{N}.\end{equation}\]

This gives the discrete time steps,

(379)¶\[\begin{equation}t_i=t_0+ih,\end{equation}\]

where \(t_0=a.\)

Here the interval is \(0≤t≤2\) and number of step 4

(380)¶\[\begin{equation}h=\frac{2−0}{4}=0.5.\end{equation}\]

This gives the discrete time steps,

(381)¶\[\begin{equation}t_i=0+i0.5,\end{equation}\]

for \(i=0,1,⋯,4.\)

# Start and end of interval
b=2
a=0
# Step size
N=4
h=(b-a)/(N)
t=np.arange(a,b+h,h)

2-step Adams Moulton¶

The general 2-step Adams Moulton difference equation is

(383)¶\[\begin{equation}w_{i+1} = w_{i} + \frac{h}{12}(5f(t_{i+1},w_{i+1})+8f(t_{i},w_{i})-f(t_{i-1},w_{i-1})). \end{equation}\]

For the specific intial value problem the 2-step Adams Moiulton difference equation is

(384)¶\[\begin{equation}w_{i+1} = w_{i} + \frac{h}{12}(5(t_{i+1}-w_{i+1})+8(t_{i}-w_{i})-(t_{i-1}-w_{i-1})). \end{equation}\]

for \(i=0\) the difference equation is:

(385)¶\[\begin{equation}w_{1} = w_{0} + \frac{h}{12}(5(t_{1}-w_{1})+8(t_{0}-w_{0})-(t_{-1}-w_{-1})).\end{equation}\]

this is not solvable as \(w_{1}, \ w_{-1}\) are unknown. for \(i=1\) the difference equation is:

(386)¶\[\begin{equation}w_{2} = w_{1} + \frac{h}{12}(5(t_{2}-w_{2})+8(t_{1}-w_{1})-(t_{0}-w_{0})). \end{equation}\]

this is not solvable as \(w_{1}\) and \(w_{2}\) are unknown. \(w_1\) can be approximated using a one step method. Here, as the exact solution is known,

(387)¶\[\begin{equation}w_1=2e^{-t_1}+t_1-1.\end{equation}\]

As the intial value problem is linear the difference equation can be rearranged such that \(w_2\) is on the right hand side:

(388)¶\[\begin{equation}w_{2}+\frac{5h}{12}w_{2} = w_{1} + \frac{h}{12}(5(t_{2})+8f(t_{1}-w_{1})-(t_{0}-w_{0})), \end{equation}\]
(389)¶\[\begin{equation}w_{2} = \frac{w_{1} + \frac{h}{12}(5(t_{2})+8f(t_{1}-w_{1})-(t_{0}-w_{0}))}{1+\frac{5h}{12}}. \end{equation}\]
### Initial conditions
w=np.zeros(len(t))
w[0]=IC
w[1]=y[1] # NEEDED FOR THE METHOD

for k in range (1,N):
    w[k+1]=(w[k]+h/12.0*(5*t[k+1]+8*myfun_ty(t[k],w[k])-myfun_ty(t[k-1],w[k-1])))/(1+5*h/12)

Plotting solution¶

def plotting(t,w,y):
    fig = plt.figure(figsize=(10,4))
    plt.plot(t,y, 'o-',color='black',label='Exact')
    plt.plot(t,w,'s:',color='blue',label='Adams-Moulton')
    plt.xlabel('time')
    plt.legend()
    plt.show

Local Error¶

The Error for the 2 step Adams Moulton is:

(390)¶\[\begin{equation}y_{n+1}=y_n+\frac{h}{12}[5f(t_{n+1},w_{n+1})+8f(t_{n},w_{n})-f(t_{n-1},w_{n-1})] +\frac{-h^4}{24}y^{(4)}(\eta),\end{equation}\]

where \(\eta \in [t_{n-1},t_{n+1}]\).

Rearranging the equations gives

(391)¶\[\begin{equation}\frac{y_{n+1}-y_{n}}{h}=\frac{1}{12}[5f(t_{n+1},w_{n+1})+8f(t_{n},w_{n})-f(t_{n-1},w_{n-1})] +\frac{-h^3}{24}y^{(4)}(\eta),\end{equation}\]

For our specific initial value problem the error is of the form:

(392)¶\[\begin{equation}\frac{-h^3}{24}y'''(\eta)=\frac{h^3}{24}2e^{-\eta} \leq\frac{(0.3)^3}{24} 2\leq 0.01042.\end{equation}\]
