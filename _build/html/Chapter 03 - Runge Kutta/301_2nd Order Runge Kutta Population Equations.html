Application of 2nd order Runge Kutta to Populations Equations¶

This notebook implements the 2nd Order Runge Kutta method for three different population intial value problems.

2nd Order Runge Kutta¶

The general 2nd Order Runge Kutta method for to the first order differential equation

(271)¶\[\begin{equation} y^{'} = f(t,y) \end{equation}\]

numerical approximates \(y\) the at time point \(t_i\) as \(w_i\) with the formula:

(272)¶\[\begin{equation} w_{i+1}=w_i+\frac{h}{2}\big[k_1+k_2],\end{equation}\]

for \(i=0,...,N-1\), where

(273)¶\[\begin{equation}k_1=f(t_i,w_i),\end{equation}\]

and

(274)¶\[\begin{equation}k_2=f(t_i+h,w_i+hk_1),\end{equation}\]

and \(h\) is the stepsize.

To illustrate the method we will apply it to three intial value problems:

1. Linear¶

Consider the linear population Differential Equation

(275)¶\[\begin{equation} y^{'}=0.1y, \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(276)¶\[\begin{equation}y(2000)=6.\end{equation}\]

2. Non-Linear Population Equation¶

Consider the non-linear population Differential Equation

(277)¶\[\begin{equation} y^{'}=0.2y-0.01y^2, \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(278)¶\[\begin{equation}y(2000)=6.\end{equation}\]

3. Non-Linear Population Equation with an oscillation¶

Consider the non-linear population Differential Equation with an oscillation

(279)¶\[\begin{equation} y^{'}=0.2y-0.01y^2+\sin(2\pi t), \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(280)¶\[\begin{equation}y(2000)=6.\end{equation}\]

Discrete Interval¶

The continuous time \(a\leq t \leq b \) is discretised into \(N\) points seperated by a constant stepsize

(281)¶\[\begin{equation} h=\frac{b-a}{N}.\end{equation}\]

Here, the interval is \(2000\leq t \leq 2020,\)

(282)¶\[\begin{equation} h=\frac{2020-2000}{200}=0.1.\end{equation}\]

This gives the 201 discrete points:

(283)¶\[\begin{equation} t_0=2000, \ t_1=2000.1, \ ... t_{200}=2020. \end{equation}\]

This is generalised to

(284)¶\[\begin{equation} t_i=2000+i0.1, \ \ \ i=0,1,...,200.\end{equation}\]

The plot below shows the discrete time steps:

t_end=2020.0

h=((t_end-t_start)/N)
t=np.arange(t_start,t_end+h/2,h)
fig = plt.figure(figsize=(10,4))
plt.plot(t,0*t,'o:',color='red')
plt.title('Illustration of discrete time points for h=%s'%(h))
plt.show()

1. Linear Population Equation¶

Exact Solution¶

The linear population equation

(285)¶\[\begin{equation} y^{'}=0.1y, \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(286)¶\[\begin{equation} y(2000)=6.\end{equation}\]

has a known exact (analytic) solution

(287)¶\[\begin{equation} y=6e^{0.1(t-2000)}. \end{equation}\]

Specific 2nd Order Runge Kutta¶

To write the specific 2nd Order Runge Kutta method for the linear population equation we need

(288)¶\[\begin{equation}f(t,y)=0.1y.\end{equation}\]
def linfun(t,w):
    ftw=0.1*w
    return ftw

this gives

(289)¶\[\begin{equation}k_1=f(t_i,w_i)=0.lw_i,\end{equation}\]
(290)¶\[\begin{equation}k_2=f(t_i+h,w_i+hk_1)=0.1(w_i+hk_1),\end{equation}\]

and the difference equation

(291)¶\[\begin{equation}w_{i+1}=w_{i}+\frac{h}{2}(k_1+k_2),\end{equation}\]

for \(i=0,...,199\), where \(w_i\) is the numerical approximation of \(y\) at time \(t_i\), with step size \(h\) and the initial condition

(292)¶\[\begin{equation}w_0=6.\end{equation}\]
w=np.zeros(N+1)
w[0]=6.0
## 2nd Order Runge Kutta
for k in range (0,N):
    k1=linfun(t[k],w[k])
    k2=linfun(t[k]+h,w[k]+h*k1)
    w[k+1]=w[k]+h/2*(k1+k2)

Plotting Results¶

y=6*np.exp(0.1*(t-2000))
fig = plt.figure(figsize=(8,4))
plt.plot(t,w,'o:',color='purple',label='Runge Kutta')
plt.plot(t,y,'s:',color='black',label='Exact')
plt.legend(loc='best')
plt.show()

2. Non-Linear Population Equation¶

(293)¶\[\begin{equation} y^{'}=0.2y-0.01y^2, \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(294)¶\[\begin{equation}y(2000)=6.\end{equation}\]

Specific 2nd Order Runge Kutta for the Non-Linear Population Equation¶

To write the specific 2nd Order Runge Kutta method we need

(295)¶\[\begin{equation}f(t,y)=0.2y-0.01y^2,\end{equation}\]

this gives

(296)¶\[\begin{equation}k_1=f(t_i,w_i)=0.2w_i-0.01w_i^2,\end{equation}\]
(297)¶\[\begin{equation}k_2=f(t_i+h,w_i+hk_1)=0.2(w_i+hk_1)-0.01(w_i+hk_1)^2,\end{equation}\]

and the difference equation

(298)¶\[\begin{equation}w_{i+1}=w_{i}+\frac{h}{2}(k_1+k_2),\end{equation}\]

for \(i=0,...,199\), where \(w_i\) is the numerical approximation of \(y\) at time \(t_i\), with step size \(h\) and the initial condition

(299)¶\[\begin{equation}w_0=6.\end{equation}\]
def nonlinfun(t,w):
    ftw=0.2*w-0.01*w*w
    return ftw

w=np.zeros(N+1)
w[0]=6.0
## 2nd Order Runge Kutta
for k in range (0,N):
    k1=nonlinfun(t[k],w[k])
    k2=nonlinfun(t[k]+h,w[k]+h*k1)
    w[k+1]=w[k]+h/2*(k1+k2)

3. Non-Linear Population Equation with an oscilation¶

(300)¶\[\begin{equation} y^{'}=0.2y-0.01y^2+\sin(2\pi t), \ \ (2000 \leq t \leq 2020), \end{equation}\]

with the initial condition,

(301)¶\[\begin{equation}y(2000)=6.\end{equation}\]

Specific 2nd Order Runge Kutta for the Non-Linear Population Equation with an oscilation¶

To write the specific 2nd Order Runge Kutta difference equation for the intial value problem we need

(302)¶\[\begin{equation}f(t,y)=0.2y-0.01y^2+\sin(2\pi t),\end{equation}\]

which gives

(303)¶\[\begin{equation}k_1=f(t_i,w_i)=0.2w_i-0.01w_i^2+\sin(2\pi t_i),\end{equation}\]
(304)¶\[\begin{equation}k_2=f(t_i+h,w_i+hk_1)=0.2(w_i+hk_1)-0.01(w_i+hk_1)^2+\sin(2\pi (t_i+h)),\end{equation}\]

and the difference equation

(305)¶\[\begin{equation}w_{i+1}=w_{i}+\frac{h}{2}(k_1+k_2),\end{equation}\]

for \(i=0,...,199\), where \(w_i\) is the numerical approximation of \(y\) at time \(t_i\), with step size \(h\) and the initial condition

(306)¶\[\begin{equation}w_0=6.\end{equation}\]
def nonlin_oscfun(t,w):
    ftw=0.2*w-0.01*w*w+np.sin(2*np.math.pi*t)
    return ftw

w=np.zeros(N+1)
w[0]=6.0
## 2nd Order Runge Kutta
for k in range (0,N):
    k1=nonlin_oscfun(t[k],w[k])
    k2=nonlin_oscfun(t[k]+h,w[k]+h*k1)
    w[k+1]=w[k]+h/2*(k1+k2)

Results¶

The plot below shows the 2nd order Runge Kutta numerical approximation, \(w\) (circles) for the non-linear population equation:

fig = plt.figure(figsize=(8,4))
plt.plot(t,w,'o:',color='purple',label='Taylor')
plt.legend(loc='best')
plt.show()
